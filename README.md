# RPG-Battle-system

Model de lupta similar cu jocurile RPG unde eroul poate ataca cu un atac simplus(baseATK) , poate folosi magie si potiun pentru vindecare.
Sistemul functioneaza  pe baza faptului ca eroul alege o optiune de atac iar inamicul reactioneaza reciproc.
# 3 inamici dintre care poti alege :Goblin , Gnome si Bossul, fiecare avand viata si putere de atac diferita.
# Scopul jocului e de a infrange inamicul prin a ii aduce viata la 0
# Magia are un sistsem de mana si e limitat de cate ori poti folosi magie
#  La fel si la potiuni
# Jucatorul joaca din prisma eroului Saphir
